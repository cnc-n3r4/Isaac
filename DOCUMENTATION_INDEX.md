## üöÄ Quick Reference Cards

### Essential Commands
```bash
/help              # Show all commands
/status            # System status
/config            # Configuration
/ask <question>    # AI questions
/workspace create  # Create workspace
/alias --enable    # Unix on Windows
isaac <query>      # Natural language
```

### File Operations
```bash
/read file.txt               # Read file
/write file.txt "content"    # Write file
/edit file.txt "old" "new"   # Edit file
/grep "pattern" **/*.py      # Search files
/glob "**/*.md"              # Find files
/newfile script.py           # Create file
```

### AI Features
```bash
/ask what is Docker?              # Information
isaac show me all python files    # Command
/analyze code.py                  # Analysis
/summarize README.md              # Summary
```

### Workspaces
```bash
/workspace create myproject --venv --collection
/workspace list
/workspace switch myproject
/workspace delete myproject
```
